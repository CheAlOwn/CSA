# ИНСТРУКЦИЯ К РАЗВЕРТЫВАНИЮ ИС

Перед развертыванием ИС необходимо установить JDK 21.0.11 и PostgreSQL 

## 1. База данных
Выгрузить базу данных в СУБД PostgreSQL.
Создать пользователя с именем app_user и выдать ему права на CRUD операции в этой БД.

## 2. Переменные окружения
Задать глобальные переменные окружения APP_ENCRYTION_KEY и APP_USER. Для APP_USER следует указать тот пароль, который указывался при создании пользователя app_user в БД

## 3. Программа
Разархивировать проект с информационной системой и создать ярлык для исполняемого файла CSA.jar. Ярлык можно переместить на рабочий стол для удобного запуска приложения
